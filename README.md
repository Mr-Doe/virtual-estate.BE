## 가상부동산

실제 대한민국 행정 구역을 지도위에 구획하여 일정 크기 블록으로 나누어 토지 분양을 합니다. <br>
블록들은 GPS 좌표 정보 데이터를 기반으로 설계하고, 관리자가 행정 동/읍으로 분양지를 생성합니다. <br>
분양을 시작하면 신청을 받을 수 있고, 신청 마감일에 인원을 추첨하여 당첨/탈락을 결정합니다. <br>
분양받으면 블록의 소유권을 가질 수 있고,  신청 포인트를 반납합니다.
